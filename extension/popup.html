AI Server Setup
The AI features require a local server and a Google Gemini API key.

1. Get a Gemini API key (free tier available):
Go to aistudio.google.com → Get API Key → Create. Paste it in the "API Key" field above.

2. Install Node.js (if you don't have it):
Download from nodejs.org (LTS version recommended).

3. Start the server — open a terminal in the extension folder and run: cd server && npm install && npm start The server runs on port 3456. Keep the terminal open while using AI features.

What the AI does:
• Scores each listing (1-10) based on your profile
• Generates personalized messages per landlord
• Solves captchas automatically
